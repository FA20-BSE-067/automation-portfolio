# Automation Starter Pack

This repository contains ready-to-use automation scripts that you can use for freelance projects. Each script is designed to save time and automate repetitive tasks.

## Scripts

1. **Google Sheets Email Automation** - Automatically send reminder emails.
2. **Python Product Scraper** - Scrape product names and prices from websites.
3. **AI Social Media Captions** - Prompt template for generating social media content using ChatGPT.
4. **Google Sheets Dashboard Automation** - Generate charts and summary reports automatically.
5. **AI Email Response Generator** - Prompt template for drafting email responses using ChatGPT.

## Usage

- Scripts 1 & 4: Upload to Google Sheets → Extensions → Apps Script.
- Script 2: Run using Python + BeautifulSoup.
- Scripts 3 & 5: Use prompts in ChatGPT to generate content.
